# OSCAR Dashboard

### Real-Time Load Cell Monitoring. Instant Reports. Zero Hassle.

---

**Stop juggling spreadsheets, manual logs, and disconnected tools.** OSCAR Dashboard brings your entire load testing workflow into one powerful desktop app.

### What it does

- **Live Monitoring** -- Connect T24 wireless load cells via USB and see real-time data with interactive graphs
- **One-Click Reports** -- Generate professional PDF and Excel reports with charts, statistics, and compliance info
- **AI-Powered Standards Lookup** -- Automatically identifies applicable ASME, OSHA, ASTM, and ISO standards for your test
- **SharePoint Integration** -- Pull job and project data directly from SharePoint -- no copy-pasting
- **CSV Import & Analysis** -- Bring in historical data for side-by-side comparison and trend analysis
- **Built-in Unit Converter** -- Quickly convert between lbs, kg, tons, and more

### Why you'll want it

| Before OSCAR | With OSCAR |
|---|---|
| Manually recording load cell readings | Live data capture with automatic logging |
| Searching for applicable standards | AI recommends the right standards instantly |
| Building reports in Excel by hand | One-click export to PDF/Excel with graphs |
| Switching between 4+ tools per job | Everything in one window |

### Get started

Download, plug in your T24 dongle, and you're running. Configure your SharePoint and OpenAI credentials in Settings for the full experience.
